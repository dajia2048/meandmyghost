🌊 关于小河流水

受 flomo、Memos、Simplenote 等优秀工具启发，但走自己的路。

核心理念：每一条想法都是一滴水，汇入你的记忆之河。

数据存储：
• 本地模式：数据保存在浏览器中
• GitHub 模式：数据同步到你的 GitHub Gist（推荐）

快捷键：
• Ctrl + Enter — 快速记录
• #标签 — 为笔记分类

建议定期导出备份，数据安全第一。

⚙️ GitHub 云同步设置

将数据同步到你的 GitHub Gist，实现跨设备访问。

  1. 打开 GitHub Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. 点击 Generate new token (classic)
  3. 勾选 gist 权限，生成 Token
  4. 把 Token 粘贴到下方

Token 只保存在你的浏览器中，不会上传到任何地方。
